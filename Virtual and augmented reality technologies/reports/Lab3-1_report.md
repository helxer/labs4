## 2. Опис головної сцени

Тематика проєкту — **інтерактивна візуалізація коливань маятника**. Головна сцена `01_SimplePendulum.unity` демонструє математичний маятник, який вільно коливається у вертикальній площині XY.

Фізична модель:
- довжина підвісу L = 2 м;
- початковий кут відхилення θ₀ = 30° від вертикалі;
- маса вантажу m = 1 кг;
- гравітація g = 9.81 м/с² (стандартна Unity);
- коефіцієнт лінійного демпфування 0.05.

Теоретичний період малих коливань:
T = 2π√(L/g) = 2π√(2/9.81) ≈ 2.84 с.

## 3. Структура сцени

| `PeriodTrigger` | Transform, MeshFilter, MeshRenderer, **BoxCollider (isTrigger)** | Зона у точці рівноваги для вимірювання періоду |

## 4. Використані компоненти фізики

### 4.1 Rigidbody (на Bob)
- `mass` = 1 кг
- `useGravity` = true
- `linearDamping` = 0.05 (для згасання коливань)
- `angularDamping` = 0.05
- `collisionDetectionMode` = Continuous

### 4.2 Colliders
- **SphereCollider** (Bob) — фізичне тіло маятника.
- **BoxCollider** (Anchor) — лише візуальне відображення пивоту.
- **MeshCollider** (Floor) — поверхня для статичної підлоги.

### 4.3 Trigger
- **BoxCollider isTrigger=true** на `PeriodTrigger`, розміщений у точці рівноваги (0, 2, 0). У Лаб. 5-2 буде використаний для підрахунку періоду (`OnTriggerEnter`).

### 4.4 Joint
- **HingeJoint** на Bob:
  - `anchor` = (–1, 1.732, 0) — локальна точка на сфері, що зв'язана з пивотом;
  - `connectedAnchor` = (0, 4, 0) — світова точка пивоту;
  - `axis` = (0, 0, 1) — маятник коливається у площині XY;
  - `autoConfigureConnectedAnchor` = false;
  - без spring/motor/limits.
- HingeJoint реалізує ідеальну механічну в'язь з однією обертовою ступенем свободи.

## 6. Результати

Після натискання Play маятник починає вільні коливання у площині XY. HingeJoint фіксує bob на відстані 2 м від точки (0, 4, 0). Коливання згасають за рахунок лінійного демпфування Rigidbody. Тригерна зона `PeriodTrigger` готова для прив'язки логіки вимірювання періоду в наступних лабораторних роботах.
