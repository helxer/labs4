## 2. Розширення сцени

У сцену `01_SimplePendulum.unity` додано два нових елементи:

### 2.1 Observer (CharacterController)

Капсула, яка моделює спостерігача у сцені. Видалено стандартний CapsuleCollider (його роль виконує сам CharacterController).

| Властивість | Значення |
|-------------|----------|
| Position | (–3, 1, –3) |
| `CharacterController.height` | 2 |
| `CharacterController.radius` | 0.5 |
| `CharacterController.center` | (0, 0, 0) |
| `CharacterController.slopeLimit` | 45° |
| `CharacterController.stepOffset` | 0.3 |

CharacterController буде використаний у Лаб. 5-2 для додавання скрипту переміщення (WASD + миша) — це дозволить розглядати маятник з різних ракурсів.

### 2.2 Другий маятник (PendulumRoot2)

Другий математичний маятник для **порівняння періодів** маятників різної довжини. Розташований поряд з першим на висоті того ж пивоту (Y = 4), зі зміщенням по X.

| GameObject | Position (world) | Параметри |
|------------|------------------|-----------|
| `PendulumRoot2` | (3, 4, 0) | Точка пивоту |
| `PendulumRoot2/Anchor2` | (3, 4, 0) | Куб 0.3 |
| `PendulumRoot2/Bob2` | (3.5, 3.134, 0) | Сфера 0.3, **Rigidbody** (m=0.5 кг), **SphereCollider**, **HingeJoint** |

HingeJoint на Bob2:
- `anchor` (local) = (–0.5, 0.866, 0)
- `connectedAnchor` (world) = (3, 4, 0)
- `axis` = (0, 0, 1)
- Початковий кут θ₀ = 30°
- Довжина L = 1 м

## 3. Порівняння теоретичних періодів

| Маятник | L | m | Теоретичний період T = 2π√(L/g) |
|---------|---|---|---------------------------------|
| Bob (перший)  | 2 м | 1.0 кг | ≈ 2.84 с |
| Bob2 (другий) | 1 м | 0.5 кг | ≈ 2.01 с |

Демонстрація демонструє ключовий фізичний факт: **період математичного маятника не залежить від маси, а залежить лише від довжини підвісу та прискорення вільного падіння**. Це наочно видно при одночасному запуску обох маятників — коротший маятник коливається помітно частіше.
