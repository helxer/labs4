# Лабораторна робота №2

**Дисципліна:** Технології віртуальної та доповненої реальності
**Тема:** Розробка моделі ПЗ для візуалізації фізичного закону «Коливання маятника» за технологією AR/VR
**Виконав:** студент групи 12-441 Рудчук Максим Олегович

---

## 1. Завдання

1. Вибрати тематику ПЗ — візуалізація фізичного закону.
2. Сформулювати вимоги до ПЗ.
3. Розробити сценарій роботи ПЗ.
4. Спроектувати ПЗ.
5. Опрацювати бібліотеки програмних модулів Unity.

## 2. Вибір теми

Тема: **Інтерактивна візуалізація коливань маятника** (математичний маятник, фізичний маятник, пружинний маятник, маятник Ньютона).

Обґрунтування:
- Дозволяє продемонструвати усі ключові компоненти фізики Unity (Rigidbody, Collider, Trigger, HingeJoint, SpringJoint).
- Зрозуміла навчальна цінність: візуалізація залежностей T = 2π√(l/g), T = 2π√(m/k), згасання, передача імпульсу.
- Підходить для публікації у вигляді WebGL-збірки (вимога лаб. 5-1).

## 3. Функціональні вимоги

| № | Вимога |
|---|--------|
| F1 | Користувач запускає математичний маятник з заданим початковим кутом |
| F2 | Параметри маятника (довжина нитки, маса вантажу, гравітація, демпфування) змінюються через UI |
| F3 | На екрані відображається поточний період коливань T і порівнюється з теоретичним T = 2π√(l/g) |
| F4 | Сцена «Пружинний маятник» демонструє вертикальні коливання вантажу на пружині (SpringJoint) |
| F5 | Сцена «Маятник Ньютона» демонструє пружні зіткнення 5 кульок на нитках |
| F6 | Доступне меню вибору сцени і кнопка «Скидання» |
| F7 | Звукові ефекти: цокання при проходженні рівноваги, дзвінок при максимальному відхиленні |
| F8 | Збірка публікується як WebGL і відкривається в браузері |

## 4. Нефункціональні вимоги

- Версія Unity: 6 LTS (6000.4.x), мова скриптів: C#.
- Цільова платформа: WebGL.
- Продуктивність: ≥ 60 FPS на десктопному браузері.
- Час холодного завантаження WebGL: ≤ 10 с.

## 5. Сценарій роботи ПЗ

1. Користувач відкриває WebGL-збірку → завантажується **головне меню**.
2. Обирає одну з трьох сцен: *Математичний маятник*, *Пружинний маятник*, *Маятник Ньютона*.
3. На обраній сцені:
   - переглядає 3D-візуалізацію зі стартовими параметрами;
   - змінює слайдерами параметри (довжина, маса, демпфування);
   - натискає **Start** → бачить рух у реальному часі та лічильник періоду;
   - порівнює виміряний період із теоретичним значенням;
   - натискає **Reset** для повернення.
4. У будь-який момент може повернутися у меню.

## 7. Використовувані компоненти Unity

| Компонент | Де використовується |
|-----------|---------------------|
| **Rigidbody** | вантажі усіх маятників |
| **Collider** (Sphere/Capsule) | кульки маятника Ньютона, вантажі |
| **Trigger** | зона на рівновазі для вимірювання періоду |
| **HingeJoint** | вісь обертання простого маятника, маятник Ньютона |
| **SpringJoint** | пружинний маятник |
| **CharacterController** | спостерігач від першої особи (опційно) |
| **Animator + Animation Clip** | анімація вступу/меню |
| **Particle System** | ефект при максимальному відхиленні |
| **AudioSource / AudioListener** | звуки |
| **Canvas + Slider/Button** | UI |

## 9. Висновок

Спроектовано модель ПЗ для AR/VR візуалізації фізичного закону коливання маятника. Визначено функціональні вимоги, сценарій взаємодії, архітектуру з трьох робочих сцен + меню, перелік компонентів Unity та програмних бібліотек. Модель забезпечує демонстрацію усіх обов'язкових елементів фізики (Rigidbody, Collider, Trigger, Joints) у лабораторних №3 та №4 і відповідає вимогам WebGL-публікації лаб. №5.
